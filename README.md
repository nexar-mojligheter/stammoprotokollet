# Stämmoprotokollet — Veckans möjlighet (Nexar Academy)

Generera juridiskt korrekta BRF-stämmoprotokoll på minuter – inte timmar.. En **researchad marknadsmöjlighet** redo att bygga.

## Öppna i Claude Code

**Desktop-app** (klistra in i webbläsarens adressfält):
```
claude://code/new?folder=~%2FNexarProjekt%2Fstammoprotokollet&q=K%C3%B6r%20exakt%20detta%20i%20mappen%3A%20git%20clone%20https%3A%2F%2Fgithub.com%2Fnexar-mojligheter%2Fstammoprotokollet%20.%20(klona%20allts%C3%A5%20in%20i%20den%20H%C3%84R%20tomma%20mappen%2C%20punkten%20p%C3%A5%20slutet).%20L%C3%A4s%20sedan%20CLAUDE.md%20och%20k%C3%B6r%20skillen%20%2Fstart%20%E2%80%94%20led%20mig%20steg%20f%C3%B6r%20steg%2C%20jag%20%C3%A4r%20nyb%C3%B6rjare.
```

**Terminal:**
```bash
git clone https://github.com/nexar-mojligheter/stammoprotokollet && cd stammoprotokollet && claude "Läs CLAUDE.md och kör /start — led mig steg för steg, jag är nybörjare."
```

Din Claude Code läser `CLAUDE.md` (instruktioner + projektminne). Hela marknadsresearchen finns i skillen `/affarsplan`.
